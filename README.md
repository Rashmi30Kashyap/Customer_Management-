# E-Commerce Customer Management System

A full-stack **Customer Management System** built with **Python, Streamlit, and MySQL** to manage **customers, products, orders, payments, and feedback** efficiently. The system replaces manual spreadsheets, automates workflows, and provides real-time analytics for better decision-making and operational efficiency.

---

## Features

- **User Login** with role-based access (Admin/Staff)  
- **Customer Management:** Add, view, update, delete customers and addresses  
- **Product Management:** Manage products, stock, and pricing  
- **Order Management:** Create and track orders with itemized details  
- **Payment Management:** Record and track payments securely  
- **Feedback Management:** Collect customer ratings and comments  
- **Real-Time Dashboard:** Display total customers, orders, products, and top-selling items  

---

## Technologies Used

- **Python** – backend logic and scripting  
- **Streamlit** – interactive frontend dashboard  
- **MySQL** – database management and storage  
- **Pandas & NumPy** – data manipulation  
- **SQL** – CRUD queries and data handling  
- **Data Visualization** – charts and metrics in dashboard  

---

## Database Structure

- **customers:** Stores customer info (name, email, phone, status)  
- **addresses:** Customer addresses linked to `customers` table  
- **products:** Product catalog with stock and pricing details  
- **orders:** Tracks customer orders and statuses  
- **order_items:** Details of products in each order  
- **payments:** Payment records linked to orders  
- **feedback:** Customer ratings and comments for products  
- **users:** Admin/Staff credentials and roles  

---

## Usage

- Login as Admin or Staff
- Navigate through the sidebar to manage customers, products, orders, payments, and feedback
- View real-time metrics and charts on the dashboard
- Add, update, and delete records directly from the interface
